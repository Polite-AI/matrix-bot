# Matrix Bot

## What is this?
See [the website](http://polite.ai)

## How do I run it?
You will need to fill in config.js for your bot to work.

### Dependencies
* NodeJS >=8.0.0
* NPM >= 5.3.0

### Development
```bash
> npm i # Install dependencies
> npm start # Start the bot
```

### Production
```bash
> npm i # Install dependencies
> npm run production # PM2
```
